# 🐍 Snake Game

A modern and responsive Snake Game built using **HTML**, **CSS**, and **JavaScript**.

## 🌐 Live Demo

👉 https://saurav56295.github.io/snake-game/

---

## ✨ Features

- 🐍 Classic Snake Gameplay
- 🌍 Wrap Around Walls
- 🍎 Red Food (+1 Score)
- 🔵 Blue Bonus Food (+5 Score)
- ⏱️ Bonus Food disappears after 5 seconds
- ⚙️ Settings Panel
- 🚀 Adjustable Snake Speed
- 🔊 Sound Toggle
- 📱 Mobile Controls
- ⌨️ Keyboard Controls
- 🏆 High Score Saved using Local Storage
- 📱 Responsive Design

## 🎮 Controls

| Key | Action |
|------|--------|
| ⬆️ | Move Up |
| ⬇️ | Move Down |
| ⬅️ | Move Left |
| ➡️ | Move Right |
| Space | Pause / Resume |
