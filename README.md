# 🐍 Snake Game

A modern Snake Game developed using HTML5, CSS3 and JavaScript.

## ✨ Features

- 🐍 Smooth Snake Movement
- 🌍 Wrap Around Walls
- 🍎 Red Food (+1 Score)
- 💎 Bonus Food (+5 Score)
- ⏱ Bonus Food Timer
- ⚙ Settings Panel
- 🎚 Adjustable Snake Speed
- 🔊 Sound Toggle
- 📱 Mobile Controls
- 🏆 High Score System
- 💻 Responsive Design

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Canvas API
- Local Storage

## 🎮 Controls

### Desktop

- ⬆ Up
- ⬇ Down
- ⬅ Left
- ➡ Right
- Space = Pause

### Mobile

Use the on-screen D-Pad.
